CSS syntax ประกอบด้วยอะไรบ้าง Selector, Declaration
การเขียน CSS มีกี่แบบอะไรบ้าง Inline CSS, Internal CSS, External CSS
selector มีกี่แบบอะไรบ้าง Simple selectors, Combinator selectors, Pseudo-class selectors, Pseudo-elements selectors, Attribute selectors
combinator มีกี่แบบอะไรบ้าง Descendant Selector, Child selector, Adjacent sibling selector, General sibling selector
